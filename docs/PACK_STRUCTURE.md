GARP Receipt Conformance Pack Structure
=======================================

Status: skeleton only.

Intended directories:

schemas/
  Future location for the canonical public receipt schema artifact and pinned digest/version marker.

fixtures/valid/
  Future location for at least one valid public receipt fixture.

fixtures/invalid/
  Future location for at least one invalid public receipt fixture.

expected/
  Future location for expected validator pass/fail outputs.

validator/
  Future location for validator entrypoints or documentation.

docs/
  Explanatory verifier documents, attestation limits, fixture provenance rules, and body_kind extension rules.

Non-claim
---------

This file describes intended structure only. It does not publish a schema, implement a validator, or validate any receipt.
